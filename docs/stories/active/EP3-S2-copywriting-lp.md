# Story EP3-S2 — Copywriting da LP
**Epic:** EP3 — Sistema de Landing Pages
**Prioridade:** P2
**Responsável:** @analyst
**Status:** Done
**Estimativa:** 1 sprint

---

## Contexto

Com o briefing coletado, o próximo passo é transformar as informações em copy persuasivo para a landing page: headline, subheadline, benefícios, CTA e prova social.

---

## User Story

**Como** gestor da Escalando Premoldados,
**quero** um processo padronizado de copywriting para LPs,
**para que** cada LP tenha textos persuasivos e consistentes sem depender de freelancer.

---

## Acceptance Criteria

- [ ] Template de copy com seções: headline, subheadline, problema, solução, benefícios (3-5), prova social, CTA
- [ ] Copy gerado a partir do briefing (pode usar IA como apoio)
- [ ] Revisão e aprovação do cliente antes do desenvolvimento
- [ ] Variações de headline para teste A/B (mínimo 2)

---

## Tarefas Técnicas

- [x] Criar template de copy (Notion ou Google Docs)
- [x] Definir prompt padrão para geração com IA (`scripts/gerar-copy.js`)
- [x] Processo de revisão com cliente (documentado no playbook)
- [x] Documentar no playbook (`docs/playbooks/copywriting-lp.md`)

---

## Definition of Done

- [ ] Copy aprovado pelo cliente antes de iniciar dev — tarefa humana
- [x] Template documentado e replicável (`docs/playbooks/copywriting-lp.md`)
- [x] Pelo menos 1 LP com copy gerado via processo ({cliente})
