# Story EP3-S3 — Template Base HTML/CSS
**Epic:** EP3 — Sistema de Landing Pages
**Prioridade:** P2
**Responsável:** @ux-design-expert + @dev
**Status:** Done
**Estimativa:** 1 sprint

---

## Contexto

Criação do template base HTML/CSS que serve como estrutura para todas as LPs dos clientes, com variáveis configuráveis por cliente.

---

## User Story

**Como** desenvolvedor da Escalando,
**quero** um template HTML/CSS base reutilizável,
**para que** cada nova LP seja criada por configuração, não por código do zero.

---

## Acceptance Criteria

- [x] Template HTML semântico com seções: hero, benefícios, prova social, formulário, rodapé
- [x] CSS com variáveis de cor e tipografia configuráveis por cliente
- [x] Responsivo (mobile-first)
- [x] Formulário de captura integrado com webhook
- [x] Tempo de carregamento < 3s

---

## Definition of Done

- [x] Template funcionando com dados de exemplo
- [x] Variáveis documentadas
- [x] Pelo menos 1 LP real gerada a partir do template
