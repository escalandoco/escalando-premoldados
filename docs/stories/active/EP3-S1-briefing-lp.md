# Story EP3-S1 — Formulário de Briefing de LP
**Epic:** EP3 — Sistema de Landing Pages
**Prioridade:** P2
**Responsável:** @analyst
**Status:** Done
**Estimativa:** 1 sprint

---

## Contexto

Para gerar LPs consistentes, é necessário um formulário de briefing padronizado que capture todas as informações necessárias do cliente antes da criação.

---

## User Story

**Como** gestor da Escalando Premoldados,
**quero** um formulário de briefing padronizado para LPs,
**para que** eu possa coletar todas as informações necessárias antes de iniciar a criação.

---

## Acceptance Criteria

- [x] Formulário com campos: produto, público-alvo, diferenciais, CTA, fotos, depoimentos
- [x] Formulário acessível via link (Google Forms ou similar)
- [x] Respostas organizadas automaticamente
- [x] Checklist de validação antes de iniciar criação da LP

---

## Definition of Done

- [x] Formulário criado e testado
- [x] Link documentado no playbook de onboarding
- [x] Pelo menos 1 briefing real coletado com sucesso
