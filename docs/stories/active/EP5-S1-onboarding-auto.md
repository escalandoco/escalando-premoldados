# Story EP5-S1 — Onboarding Automático Completo
**Epic:** EP5 — Escala Multi-Cliente
**Prioridade:** P3
**Responsável:** @dev
**Status:** Pending
**Estimativa:** 2 sprints

---

## User Story

**Como** gestor da Escalando Premoldados,
**quero** um processo de onboarding totalmente automatizado para novos clientes,
**para que** eu possa ativar um novo cliente em menos de 30 minutos.

---

## Acceptance Criteria

- [ ] Pipeline: formulário de fechamento → ClickUp (folder + listas + tasks) + Git (config) automático
- [ ] LP em staging gerada automaticamente após briefing
- [ ] Planilha de leads criada com aba do cliente
- [ ] Tintim configurado para o número do cliente
- [ ] E-mail/WhatsApp de boas-vindas enviado automaticamente
- [ ] Checklist de onboarding no ClickUp 100% automatizado
