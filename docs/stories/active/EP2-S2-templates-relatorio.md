# Story EP2-S2 — Templates de Relatório de Performance
**Epic:** Epic 2 — Sistema de Leads
**Prioridade:** P1
**Responsável:** @analyst
**Status:** Pending
**Estimativa:** 1 sprint
**Depende de:** EP2-S1 (planilha como fonte de dados)

---

## Contexto

O relatório mensal é um dos entregáveis centrais dos planos. Precisa ser gerado com o mínimo de trabalho manual, ser claro para o cliente (que entende de negócio, não de marketing) e mostrar ROI — não métricas de vaidade.

---

## User Story

**Como** gestor da Escalando Premoldados,
**quero** templates de relatório mensal e quinzenal prontos para preencher,
**para que** eu possa entregar ao cliente uma visão clara de resultado sem gastar mais de 30 minutos por relatório.

---

## Acceptance Criteria

### Relatório Mensal (Plano Starter e Growth)

- [ ] **Capa:** Logo Escalando + nome do cliente + período + data de entrega
- [ ] **Resumo executivo** (1 página): investimento total, total de leads, CPL médio, leads qualificados, resultado em palavras simples ("mês positivo / abaixo do esperado / dentro da meta")
- [ ] **Meta Ads** (se ativo): investimento, alcance, cliques, leads, CPL, melhor criativo do mês
- [ ] **Google Ads** (se ativo): investimento, impressões, cliques, leads, CPL, palavras-chave com melhor desempenho
- [ ] **GMB**: visualizações do perfil, buscas, cliques para ligar, cliques para navegar
- [ ] **Leads por status**: novo, contatado, negociação, fechado, perdido (tabela simples)
- [ ] **Próximo mês**: 3 ações planejadas (o que vai mudar/testar)
- [ ] **Mensagem do gestor**: parágrafo humanizado com análise e recomendação

### Relatório Quinzenal (Plano Pro — adicional)

- [ ] Versão reduzida: investimento até a data, leads até a data, CPL parcial, 1 insight e 1 ajuste realizado
- [ ] Formato: mensagem de WhatsApp (texto + print de dashboard) ou PDF de 1 página

### Formatos

- [ ] Template Google Slides (relatório mensal) — duplicável por cliente
- [ ] Template Google Docs (alternativa textual) — para clientes que preferem
- [ ] Template de mensagem WhatsApp (quinzenal e mensal resumido)
- [ ] Todos os templates com espaços marcados [PREENCHER] claramente

### Automação AIOS (preparação)

- [ ] Mapear quais dados vêm da planilha automaticamente
- [ ] Mapear quais dados precisam ser inseridos manualmente
- [ ] Documentar prompt para AIOS gerar rascunho do relatório com dados da planilha

---

## Tarefas Técnicas

- [ ] Criar template mensal no Google Slides
- [ ] Criar template quinzenal (1 página)
- [ ] Criar template de mensagem WhatsApp (texto formatado)
- [ ] Testar preenchimento com dados reais (ou simulados)
- [ ] Cronometrar tempo de preenchimento (meta: < 30 min)
- [ ] Documentar o processo de geração do relatório

- [ ] Templates criados e acessíveis via Google Drive (link documentado)
- [ ] Tempo de preenchimento testado e < 30 minutos
- [ ] Aprovado pelo fundador com 1 relatório de teste completo
