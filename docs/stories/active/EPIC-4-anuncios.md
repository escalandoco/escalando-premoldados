## Objetivo
Implementar pipeline completo de anúncios pagos (Meta Ads + Google Ads) seguindo a metodologia Pedro Sobral: briefing estruturado → benchmarking de concorrentes → estratégia + nomenclatura → públicos personalizados → copy com 7 pilares → criativos → setup técnico (Meta + Google) → go-live → relatório de performance → escala.

## Metodologia base — Pedro Sobral

**7 Pilares do anúncio:**
1. Hook (gancho) — primeiros 3 segundos
2. Narrativa — contexto e identificação
3. Dor/Desejo — problema ou sonho do avatar
4. Contra-intuitivo — elemento de surpresa/credibilidade
5. CTA — chamada para ação clara
6. Segmentação — público certo, plataforma certa
7. Prática — testar, medir, iterar

**Fluxo de campanha:**
Briefing + Ideias → Benchmarking → Estratégia + Nomenclatura → Públicos → Copy → Criativos → Setup Meta → Setup Google → Go-Live → Relatório → Escala
