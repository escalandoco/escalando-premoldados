## Objetivo
Criar um sistema replicável para gerar, otimizar e manter landing pages customizadas por cliente e produto — desde o briefing até SEO local e testes A/B — com rastreamento completo e deploy no domínio do cliente.

- [ ] LP gerada em menos de 2h a partir do briefing

- [ ] Testes A/B rodando em pelo menos 2 LPs
- [ ] Heatmaps configurados para clientes ativos
- [ ] Todas as LPs com PageSpeed > 90
- [ ] LPs rankeando para termos locais dos clientes
