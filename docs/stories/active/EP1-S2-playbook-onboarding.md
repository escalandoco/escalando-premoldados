# Story EP1-S2 — Playbook de Onboarding
**Epic:** EP1 — Presença Digital Base
**Prioridade:** P0
**Responsável:** @sm
**Status:** Done
**Estimativa:** 1 sprint
**Depende de:** EP1-S1 (LP precisa existir para onboarding fazer sentido)

---

## Contexto

Sem processo documentado, cada novo cliente vai exigir improviso do fundador. O playbook de onboarding é o que transforma "agência de uma pessoa" em "sistema replicável". Deve cobrir as 3 semanas de onboarding e ser executável por qualquer pessoa da equipe — ou pelo AIOS.

---

## User Story

**Como** operador da Escalando Premoldados,
**quero** um playbook completo de onboarding para novos clientes,
**para que** eu possa executar os primeiros 21 dias com qualidade consistente, sem depender de memória ou improviso.

---

## Acceptance Criteria

### Checklist de Kickoff (Semana 1)

- [ ] Script de reunião de kickoff (roteiro de 1h com tópicos obrigatórios)
- [ ] Formulário de briefing (Google Forms) com campos:
  - Nome da empresa, CNPJ, responsável, WhatsApp
  - Produtos principais e diferenciais
  - Área de atuação (cidades/estados)
  - Principais clientes (perfil)
  - Ticket médio de venda
  - Como vendem hoje
  - Plano contratado (Starter/Growth/Pro)
  - Verba disponível (Pro)
- [ ] Template de mensagem para criar grupo WhatsApp
- [ ] Roteiro de captação de fotos (PDF enviado ao cliente):
  - O que fotografar: fachada, produto, obra em andamento, equipe
  - Como fotografar: luz, ângulo, fundo
  - Exemplos de boas e más fotos
  - Frequência de envio sugerida
- [ ] Checklist de acessos necessários (Meta Business, Google Ads, GMB, site)

### Setup (Semana 2)

- [ ] Checklist de configuração de LP (itens obrigatórios antes de publicar)
- [ ] Checklist de configuração GMB (15 itens de otimização)
- [ ] Checklist de configuração de campanhas (Meta e/ou Google)
- [ ] Checklist de configuração Sheets/CRM (colunas, fórmulas, permissões)
- [ ] Checklist de configuração Tintim (rastreio de leads WPP)
- [ ] Template de briefing de criativos (o que informar ao designer)
- [ ] Mensagem padrão para envio de criativos ao cliente para aprovação

### Go-live (Semana 2–3)

- [ ] Checklist de go-live (o que verificar antes de ligar campanhas)
- [ ] Mensagem padrão de notificação do primeiro lead ao cliente
- [ ] Reunião rápida de alinhamento (pauta padrão pós go-live)

### Operação Mensal (Templates)

- [ ] Template de pauta de reunião mensal
- [ ] Template de mensagem de relatório mensal (WhatsApp)
- [ ] Calendário editorial de criativos (quando pedir fotos, quando entregar artes)

### Formato de entrega

- [ ] Documento principal em Markdown (docs/playbooks/onboarding.md)
- [ ] Formulário de briefing em Google Forms (link no documento)
- [ ] Roteiro de fotos em PDF (docs/playbooks/roteiro-fotos.pdf ou .md)
- [ ] Todas as mensagens-padrão em seção "Templates de Comunicação"
