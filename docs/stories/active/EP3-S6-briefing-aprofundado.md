# Story EP3-S6 — Briefing Aprofundado (LP única por cliente)
**Epic:** EP3 — Sistema de Landing Pages
**Prioridade:** P2
**Responsável:** @ux-design-expert + @analyst
**Status:** Done
**Estimativa:** 1 sprint

---

## Contexto

Além do briefing padrão (EP3-S1), clientes no plano Growth/Pro recebem uma LP única e exclusiva, não baseada em template. Isso exige um briefing mais profundo com estratégia de posicionamento.

---

## User Story

**Como** gestor da Escalando Premoldados,
**quero** um processo de briefing aprofundado para LPs exclusivas,
**para que** clientes premium recebam uma LP estratégica e diferenciada.

---

## Acceptance Criteria

- [x] Briefing aprofundado com: pesquisa de concorrentes, posicionamento, persona detalhada, jornada do cliente
- [x] Entrevista de 30-60min com o cliente (roteiro documentado)
- [x] Documento de estratégia de LP entregue ao cliente antes do desenvolvimento
- [x] Diferenciação clara do template padrão

---

## Definition of Done

- [x] Roteiro de entrevista documentado
- [x] Template de documento de estratégia criado
- [x] Pelo menos 1 briefing aprofundado realizado
