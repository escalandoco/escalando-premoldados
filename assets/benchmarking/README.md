# assets/benchmarking/

Pasta para screenshots de anúncios coletados na Meta Ad Library.

## Nomenclatura dos arquivos

```
{slug-concorrente}_{data}_{numero}.png

Ex:
premoldados-se_2026-03_01.png
mourao-concreto-al_2026-03_01.png
```

## Como capturar

1. Abra o anúncio na Meta Ad Library
2. Screenshot da visualização completa (copy + criativo + CTA)
3. Salve aqui com a nomenclatura acima
4. Adicione a análise no playbook: `docs/playbooks/benchmarking-ads.md`
